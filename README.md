# QuickNotes-Flutter-Notes-Taking-App

QuickNotes is a lightweight Flutter note-taking app (MVP) for quickly creating, editing, deleting and searching notes offline. It stores notes locally using `shared_preferences`.

## Features
- Create, edit, delete notes
- Search notes by title/content
- Basic color tagging
- Offline-first (local storage)

## Tech
- Flutter (Dart)
- shared_preferences
- provider

## Run locally
1. `flutter pub get`
2. `flutter run`
